 Create the following Entities.

1) Employee (id, name, designation, salary,phone,email and password)
2) Department(id, name, location)

 Department and Employee have one to many Relationship Perform the following tasks
1) Save the department
2) Save the Employee
3) Update the Department
4) Update the Employee
5) Find the Employee by Department id
6) Find the Employee by Department name
7) Verify Employee by phone and Password
8) Verify Employee By Email and Password
9) Verify Employee By id and password
10) Find Department By id

NOTE : Use JPA or Hibernate, use command Line Interface, use DTO, DAO and Controller
